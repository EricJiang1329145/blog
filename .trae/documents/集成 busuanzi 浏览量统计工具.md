# 集成 busuanzi 浏览量统计工具

## 目标
在 Neon Cosmos 博客项目中集成 busuanzi 统计工具，实现网站浏览量统计功能，包括站点总访问量和文章页面访问量。

## 实施步骤

### 1. 在布局文件中添加 busuanzi 跟踪代码
- 修改 `src/app/layout.tsx` 文件
- 在 `<head>` 标签中添加 busuanzi 的 JavaScript 代码
- 确保代码只在客户端执行

### 2. 在主页添加站点统计信息
- 修改 `src/app/page.tsx` 文件
- 在 "网站数据分析" 模块中添加站点总访问量和访客数显示
- 使用 busuanzi 提供的标签结构

### 3. 在文章页面添加浏览量统计
- 修改 `src/app/posts/[slug]/page.tsx` 文件
- 在文章元信息部分添加该文章的浏览量显示
- 使用 busuanzi 提供的页面访问量标签

### 4. 样式调整
- 确保统计数据的显示与网站的霓虹主题风格一致
- 添加适当的样式，使统计数据显示美观

## 预期效果
- 网站首页显示站点总访问量和访客数
- 每篇文章页面显示该文章的浏览量
- 统计数据实时更新
- 与网站整体设计风格协调一致

## 技术实现
使用 busuanzi 提供的轻量级 JavaScript 代码，无需后端服务，完全基于前端实现，符合项目的静态网站特性。
